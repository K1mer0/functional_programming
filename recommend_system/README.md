# Рекомендательная система книг

## Что реализовано
- База книг в `data/books.json` (JSON-массив объектов).
- CLI-интерфейс:
  - ввод жанров/авторов/ключевых слов,
  - фильтры,
  - сортировки,
  - вывод,
  - выбор книг в список `прочитать`,
  - сохранение рекомендаций в JSON/CSV.
- Веб-интерфейс на Flask (`web_app.py`):
  - вывод рекомендаций,
  - выбор книг чекбоксами в список “прочитать”,
  - экспорт рекомендаций и списка “прочитать” в JSON/CSV.

## Запуск (CLI)
```bash
cd book_recommender
python3 app.py
```

## Веб-интерфейс
```bash
pip install flask
python3 web_app.py
```
Откройте `http://127.0.0.1:5000`.
